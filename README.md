# Getting Started

Web Service for parsing html pages. 

## Request example

```
curl --location --request GET 'http://localhost:8080/html?url=https://www.sitejabber.com/reviews/keh.com'
```

### Technologies

* [Vavr.io](https://docs.vavr.io/)
* Spring WebFlux
* [Jsoup](https://jsoup.org/)
* Jacoco Test Report
